# React Blog

A modern, responsive blog built with React and Vite featuring a beautiful design with custom color scheme.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with custom color scheme
- **Navigation**: Easy navigation between Home, Blog, and About pages
- **Interactive Blog**: Click on blog posts to read full content
- **Color Scheme**: 
  - Primary Green: #4C7026
  - Secondary Green: #B6CF7B
  - Primary Blue: #28448E
  - Secondary Blue: #73A4CD

## Pages

- **Home**: Welcome page with featured posts
- **Blog**: List of blog posts with detailed view
- **About**: Information about the team and mission

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Technologies Used

- React 18
- Vite
- React Router DOM
- CSS3 with CSS Grid and Flexbox
- JavaScript ES6+

## Customization

The blog is fully customizable:
- Modify colors in the CSS files
- Add new blog posts in the Blog component
- Update team information in the About component
- Customize the home page content
